# Build Process Details
- Use Webpack to bundle JS and CSS.
- Handlebars processes templates with JSON data.
- Output static HTML, CSS, JS to dist/ folder.
- GitHub Pages serves from dist/ or root depending on configuration.

### Content Schema
Define JSON schemas for projects, profile, etc., to ensure consistency.

## Best Practices and Patterns Followed

- **Modular Development:** Separate concerns (HTML, CSS, JS).
- **Version Control:** Commit frequently with descriptive messages.
- **Progressive Enhancement:** Start with core functionality, add enhancements.
- **Mobile-First Design:** Design for mobile, enhance for larger screens.
- **Semantic HTML:** Use proper elements for accessibility.
- **Performance Optimization:** Minimize assets, optimize images.
- **Security:** Avoid inline scripts, validate inputs.
- **Maintainability:** Clear naming, comments, documentation.
