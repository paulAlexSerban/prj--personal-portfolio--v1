# Personal Portfolio V1

## Overview
This is a simple, static personal portfolio website to showcase projects, coursework, and software engineering skills. It is built with HTML, CSS, JavaScript, Webpack, and Handlebars, and is hosted on GitHub Pages.

## Features
- About Me section
- Projects showcase
- Contact section with email and LinkedIn links
- Responsive design
- Accessibility improvements
- Animations for better user experience

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/paulAlexSerban/prj--personal-portfolio--v1.git
   ```
2. Navigate to the project directory:
   ```bash
   cd prj--personal-portfolio--v1
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run start
   ```
5. Build the project for production:
   ```bash
   npm run build
   ```

## Future Improvements

## Deployment
The site is automatically deployed to GitHub Pages using a GitHub Actions workflow. Any changes pushed to the `main` branch will trigger a new deployment.
