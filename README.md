# Personal Portfolio V1

A simple, static personal portfolio website to showcase projects, coursework, and software engineering skills. Built with vanilla HTML, CSS, JavaScript, Webpack, and Handlebars. Hosted on GitHub Pages.

## Features
- Project and coursework showcase
- Professional profile links (GitHub, LinkedIn, freeCodeCamp, HackerRank)
- Responsive design
- Static site generation

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Build for production: `npm run build`
